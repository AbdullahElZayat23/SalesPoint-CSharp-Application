
# Warehouse management system

An app for managing company warehouse' products.



## Authors

- [@AbdullahElZayat23](https://github.com/AbdullahElZayat23)


## Features

- Multi-warehouses
- Customers & suppliers
- Product transfer across warehouses
- Reports


## How to use

Download and run with Visual studio.
